## üöÄ Tracks

CODORRA features four distinct tracks for competition:

1. **AI + Security**: Building defensive AI systems, securing LLMs, and ML-powered threat detection.
2. **Network Security**: Infrastructure hardening, protocol security, and zero-trust architectures.
3. **Tomfoolery**: The unconventional and absurd. Stupid concept, but brilliant execution. (e.g., over-engineered digital pranks).
4. **Open Innovation**: The wildcard track for high-impact security projects that defy categorization.
